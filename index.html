Options

Variable Default Type Description
clickable true Boolean Enables/disables animation triggered on click
speed normal String Sets click animation speed.
Other options:
  • slow
  • fast
withoutX false Boolean Disables/enables element's rotation around X axis
withoutY false Boolean Disables/enables element's rotation around Y axis
